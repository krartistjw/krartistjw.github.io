# Homebrew

## Homebrew 란?
터미널 CLI(Command-line interface) 환경에서 명령어를 작성하여 손쉽게 프로그램을 설치, 업데이트, 삭제 등을 할 수 있도록 도와주는 도구이다.

## 사전정보
- Homebrew 공식 페이지: https://brew.sh/index_ko
